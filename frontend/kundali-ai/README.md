# kundali-ai
Starter pack to run an Ollama-powered Kundali (Vedic astrology) assistant.

Features:
- Modelfile configured for llama3.1:70b
- Data skeleton for planets, houses, signs, nakshatras
- Prompts (system + interpretation)
- Example scripts to generate ephemeris (Swiss Ephemeris) and query Ollama
- RAG skeleton and instructions

Language: auto-mix Nepali/English. Response style: Blend of classical Vedic + modern simplified.

Files:
- Modelfile
- prompts/
- data/
- scripts/
- vector_db/ (placeholder for Chroma/Pinecone index)
